## 👋 About this repo

This is my personal **learning log** as an AI & Data Science student at **Saylani Mass IT Training, Karachi**. Instead of hiding my notes in a private folder, I push every class notebook and practice exercise here — messy first drafts included — so the progress is visible and so future-me (and anyone else learning) can look back at exactly how this journey started.

No fluff, no "10/10 perfect portfolio" notebooks. Just real, dated, working code from real classes.

> 🎯 **Long-term goal:** Go from Python fundamentals → Pandas/Data Analysis → Machine Learning → solving real-world problems with data and eventually building/freelancing on top of these skills.

## 🚀 Running the notebooks locally

```bash
# 1. Clone the repository
git clone https://github.com/Syed-Hassaan-Bukhari/AI-AND-DATA-SCIENCE.git
cd AI-AND-DATA-SCIENCE

# 2. (Recommended) Create a virtual environment
python -m venv venv
source venv/bin/activate      # On Windows: venv\Scripts\activate

# 3. Install common dependencies
pip install pandas numpy jupyter openpyxl

# 4. Launch Jupyter and open any notebook
jupyter notebook
```
